# FileGenius Website

Welcome to the FileGenius landing page website!

## 🚀 Features

- 📱 Responsive design (mobile, tablet, desktop)
- 🌍 Multi-language support (40+ languages)
- 🎨 Multiple themes (Minimalist, Colorful, Professional)
- 🌙 Dark mode support
- ⚡ Fast and lightweight (single HTML file)
- 🔗 One-click email contact

## 📄 Pages

- **Home** - Introduction and key features
- **Features** - Detailed feature descriptions
- **Pricing** - Subscription plans
- **Download** - Download links for Linux, macOS, Windows
- **About** - About the developer
- **Contact** - Contact information
- **Privacy** - Privacy policy
- **Terms** - Terms of service
- **License** - License information

## 🌐 Live Demo

Visit the live website: [FileGenius](https://ai-file-organizer-website.vercel.app)

## 📝 Customization

To customize the website:

1. Edit `index.html` directly
2. Modify colors in the CSS section
3. Update content in the JavaScript translations object
4. Add more languages by extending the translations

## 🔧 Development

No build process required! Simply open `index.html` in your browser.

## 📦 Deployment

This website is deployed on:
- **Vercel** - Automatic deployment from GitHub
- **GitHub Pages** - Static hosting
